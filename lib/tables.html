Basic Table

Minimalist design.

# Name Position
8 Alex Ovechkin Left Wing
19 Nicklas Backstrom Center
70 Braden Holtby Goalie

A default table uses ordinary table markup: 

<table>
  <thead>
    <tr><th>#</th></tr>
    …
  </thead>
  <tbody>
    <tr><td>8</td></tr>
    …
  </tbody>
</table>
